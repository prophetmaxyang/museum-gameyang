# 枫泾小掌柜·时光信箱

一个基于 Phaser.js 的竖屏文字 RPG 游戏，讲述小女孩阿杏在枫泾古镇与乐高乐园之间迷路的故事。

## 项目结构
- `index.html` - 主页面
- `game.js` - 游戏逻辑与对话树
- `phaser.js` - Phaser 3 引擎
- `assets/` - 图片与音频资源
- `types/` - TypeScript 类型定义（可选）

## 如何运行
1. 将项目上传到 GitHub Pages / 静态托管
2. 访问 `https://prophetmaxyang.github.io/museum-game/` 即可游玩

## 功能特性
- 竖屏适配 (750×1334)
- 画框式背景 + 氛围渐变遮罩
- 立绘表情系统
- 多分支对话 (古镇/乐高/离开)
- 地图展示（牌坊路线、乐高路线、公交线路）
- 双结局（古镇横扫镜头、乐高缩放远离）
